========================================
Circuit Simulator MNA
========================================

Bem-vindo à documentação do Circuit Simulator MNA!

Este é um simulador SPICE educacional implementado em Python, utilizando o método de Análise Nodal Modificada (MNA) e Newton-Raphson para resolver circuitos lineares e não-lineares.
